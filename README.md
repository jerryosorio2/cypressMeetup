# Cypress meetup

Cypress meetup examples

## Installation

First clone repository

```bash
git clone https://github.com/jerryosorio2/cypressMeetup.git
```

Install packages

```bash
npm install
```

Execute test

```bash
npm test
```

Or to execute specific test group use the following command

```bash
npx cypress run --spec 'pathOfSpecs'
```
